:::caution[Obsoleto]
Este error proviene de una versión anterior de Astro y ya no se utiliza. Si no puedes actualizar tu proyecto a una versión más reciente, puedes consultar [Documentación antigua (sin mantenimiento)](/es/upgrade-astro/#documentación-antigua-sin-mantenimiento) para obtener ayuda.
:::

> **InvalidRewrite404**: Reescribir un 404 solo está permitido dentro de las páginas bajo demanda.

## ¿Qué salió mal?
El usuario intentó reescribir una página 404 dentro de una página estática.
